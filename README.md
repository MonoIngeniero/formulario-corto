# Formulario de Cultura General

## Descripción

Aplicación de escritorio desarrollada en **Visual Basic .NET (Windows Forms)** que permite responder un formulario interactivo de preguntas de cultura general.

El proyecto fue realizado como práctica de desarrollo de interfaces gráficas, manejo de eventos y validación de respuestas utilizando Windows Forms.

---

## Características

- Interfaz gráfica amigable.
- Preguntas de cultura general.
- Validación de respuestas.
- Cálculo del resultado obtenido.
- Navegación mediante controles de Windows Forms.

---

## Tecnologías utilizadas

- Visual Basic .NET
- Windows Forms
- Visual Studio

---

## Conceptos aplicados

- Programación orientada a eventos.
- Interfaces gráficas (GUI).
- Controles de Windows Forms.
- Validación de datos.
- Modularización del código.
